# spring-simple-board
Spring으로 구현한 게시판입니다.  
간단한 기능으로 시작해서 기능을 추가중입니다.

기능
---------

*회원가입  
*로그인(진행중)  
~*회원 마이페이지~    
*게시글 작성  
*게시글 나열  
*게시글 검색  
*게시글 삭제  
*댓글 달기(진행중)  
~*댓글 삭제~  
~*css 추가~  
  
구현 예정

----
